============
Introduction
============

The `Site Log Manager (SLM)` is a fully-functional web tool for managing GNSS_
ground station meta data. `SLM` is maintained by the IGS_ Governing Board and
is freely licensed for general use under the MIT License. The `SLM` is
implemented in Python_ and JavaScript using the Django_ web framework. A general
understanding of Django_ is a prerequisite for understanding how SLM is put
together.

You can report bugs and discuss features on the
`issues page <https://github.com/igs/slm/issues>`_.

`Contributions <https://github.com/igs/slm/blob/main/CONTRIBUTING.md>`_ from
the community are encouraged!
